
# Mayble

A book management application.

## Reason 

I wrote this software for a family member who has a collection of books and needed a way to keep track of them. What have been read, and been loaned out.

## Futures

    - Searching, and Sorting
    - Track read, and borrowed books
    - Rate books
    - CSV import / export 

## Requirements

- [The Fyne framework and it's needed dependencies.](https://docs.fyne.io/started/quick/)
- And Go 1.24

## Build

You can run `go run .` to build and run the program and try it out.

## Notice on Releases

The releases for this project are primarily for personal use and will not have any more then what is needed for the goal of this project.
